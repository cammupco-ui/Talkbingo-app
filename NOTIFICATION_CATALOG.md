## 1. 알림 타입 요약

| 타입 | 구현 방식 | 위치 | 소거 |
|---|---|---|---|
| **Center Alert** | `_showCenterNotification()` → `OverlayEntry` | 화면 중앙, 딤 없음 | 2초 자동 |
| **Notification Modal** | `_showNotificationModal()` → `showDialog` | 화면 중앙 모달 | 버튼 탭 |
| **SnackBar** | `_showSnackBar()` → `_showCenterNotification()` 위임 | 화면 중앙 | 2초 자동 |
| **Dialog Light** | `showDialog + AlertDialog` (흰 배경) | 화면 중앙 모달 | 버튼 탭 |
| **Dialog Dark** | `showDialog + AlertDialog` (`surface2`) | 화면 중앙 모달 | 버튼 탭 |
| **Loading Dialog** | `showDialog` (진행 상태 표시) | 화면 중앙 모달 | 완료 시 자동 |

> **구현 참고:** `_showSnackBar()`는 내부적으로 `_showCenterNotification()`을 호출합니다.
> `game_screen.dart`의 Center Alert / SnackBar 는 동일한 흰 카드 스타일입니다.
